## Scope

- User-level and system-level default app handling
- File extension and MIME/UTI/protocol associations
- Sandboxed app launch behavior (Flatpak/portal, AppContainer/UWP constraints)
- Enterprise/device policy paths where user defaults are constrained

## Out of Scope (for now)

- Mobile OS default app behavior (iOS/Android)
- Browser-internal link routing behavior not exposed through OS association APIs
